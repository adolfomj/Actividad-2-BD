# Proyecto Staging - Jardinería

## Descripción
Este proyecto consiste en la construcción de una base de datos Staging a partir de la base de datos Jardinería. Se realizó el proceso de extracción, transformación y carga de datos (ETL), junto con validaciones para asegurar la calidad de la información.

## Contenido
El repositorio incluye:

- Scripts SQL para la creación de la base de datos Staging  
- Scripts de creación de tablas  
- Scripts de carga de datos (ETL)  
- Scripts de validación  
- Backups de las bases de datos:
  - jardineria.bak  
  - staging.bak
